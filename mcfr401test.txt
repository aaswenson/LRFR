c The control blades in the reflector region were removedper Nick Smith's 
c remarks of having a second system that may be unnessary and prone to issues. 
c
c This input file evaluates a 14m^3 volume core with a fuel mixture of 
c 45.2 NaCl - 46 UCl3 - 8.65(TRU)Cl3. The TRU composition used was 
c provided in the REBUS-3700 design concept. The above fuel composition 
c was chosen to increase the amount of U-238 in the fuel and keep the 
c melting point below 650 C. 
c
c The geometry of the core is simply a cylinder core with a TZM vessel and 
c a 60 cm thick lead-magnesium eutectic (LME) reflector outside of the core.
c A B4C poison was added beyond the reflector to act a shield and also reduce
c the chance of criticality in the heat exchanger.
c
c There are four outlet pipes and four inlet pipes into the core filled with 
c fuel salts. There are control blades of various compositions located in the 
c reflector region.
